# 🔍 **Forensics in CTF’s**

Esta carpeta contiene mis writeups de retos de **forense digital** de picoCTF.  
Incluyen análisis de imágenes, metadatos, archivos sospechosos, cadenas ocultas y técnicas básicas de investigación.

---

## 🧠 **Qué estoy aprendiendo aquí**

- Uso de herramientas como `strings`, `exiftool`, `binwalk`, `xxd`.  
- Identificación de datos ocultos.  
- Análisis de archivos y formatos.  
- Pensamiento forense paso a paso.

---

## 🎯 **Objetivo**

Construir una base sólida en forense para avanzar hacia retos más complejos y análisis más profundos.
