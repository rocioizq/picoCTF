# Repositorio PicoCTF – Notas Personales y Soluciones

Bienvenido a mi **repositorio de PicoCTF**, donde documento mi proceso de aprendizaje a través de desafíos de Capture The Flag (CTF). Este repositorio sirve tanto como **referencia personal** como para **mostrar mis habilidades en ciberseguridad**, resolución de problemas y metodología en CTFs.

---

## **Propósito del Repositorio**

- Servir como **referencia personal** para repasar técnicas y soluciones.  
- Mostrar mis **habilidades de resolución de CTFs** a reclutadores, mentores o colaboradores.  
- Hacer seguimiento de mi **progreso a lo largo del tiempo** y reforzar el aprendizaje mediante documentación.
