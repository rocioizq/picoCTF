# 🧠 **picoCTF Writeups – Rocío Izquierdo Lammoglia**

Este repositorio documenta mi progreso resolviendo retos de **picoCTF**, como parte de mi camino de aprendizaje en **ciberseguridad**, **forense**, **criptografía**, **Linux** y **seguridad web**.

Mi objetivo no es mostrar soluciones perfectas, sino **demostrar cómo aprendo, cómo razono y cómo mejoro con cada reto**.  
Cada writeup refleja mi proceso real: qué pensé, qué probé, qué funcionó y qué aprendí.

---

# 📂 **Estructura del repositorio**

- **Forensics in CTF’s**  
  Retos de análisis de archivos, metadatos, imágenes, memoria y técnicas básicas de forense digital.

- **General Skills in CTF**  
  Retos introductorios que fortalecen fundamentos: terminal, lógica, scripting, manipulación de archivos, etc.

- **The Beginner’s Guide to the picoGym**  
  Mis primeros pasos en picoCTF, donde construí la base para retos más avanzados.
